# endpoints
- `GET /health` - Health check
- `GET /api/info` - Service information
- `GET /api/bots` - List agents
- `GET /api/runs` - Get bot runs
- `GET /api/runs/real` - Get real data
- `POST /api/evaluate` - Evaluate a run
- `GET /api/runs/compare` - Compare data
- `GET /api/statistics` - Statistics
